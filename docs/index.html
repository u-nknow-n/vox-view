❓ Perguntas Frequentes

Por que o Windows Defender alerta sobre o app?

O Vox Launcher é compilado via PyInstaller. Infelizmente, por ser um método comum de empacotamento em Python, alguns antivírus geram "falsos positivos". O projeto é Open Source e totalmente seguro. Você pode adicionar uma exceção nas configurações de segurança do Windows sem riscos.

Como gerenciar os Mods no servidor?

Você não precisa configurar nada no Launcher! Basta habilitar os mods desejados durante a criação do mundo dentro do próprio Don't Starve Together. O Launcher lê essas preferências automaticamente ao iniciar o processo.

Onde encontro o meu Cluster Token?

O token é essencial para autenticar seu servidor. Você pode gerar um novo na sua conta oficial da Klei Accounts. Sem ele, o servidor não aparecerá na lista pública do jogo.

Encontrei um erro, o que devo fazer?

Se você encontrar comportamentos inesperados, por favor, abra uma Issue no repositório oficial. Anexar o arquivo de log (localizado em appdata/logs/applog.txt) ajuda muito na resolução rápida do problema.
