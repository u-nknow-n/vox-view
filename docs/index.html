❓ Perguntas Frequentes

Por que o Windows Defender alerta sobre o app?

O Vox Launcher é compilado via PyInstaller. Infelizmente, por ser um método comum de empacotamento em Python, alguns antivírus geram "falsos positivos". O projeto é Open Source e totalmente seguro.

Como gerenciar os Mods no servidor?

Basta habilitar os mods desejados durante a criação do mundo dentro do próprio Don't Starve Together. O Launcher lê essas preferências automaticamente.

Onde encontro o meu Cluster Token?

O token é gerado na sua conta oficial da Klei Accounts. Sem ele, o servidor não aparecerá na lista pública do jogo.
